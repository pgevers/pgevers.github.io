Test Launch Announcement
PK - Mobile - Stacked YMAN
On Monday, PK launched a mobile test showing YMAN product recommendations vertically stacked, instead of on a horizontal scroll. We think this will improve user interaction with the YMAN recommendations, and will improve product discovery.

We expect users to find more relevant products with less friction, which will ultimately drive improved conversion and RPV.

Test Duration: 10/14/19-10/25/19

(~9 days)
